# GreenBite Café

Landing page ficticia desarrollada en React para el proyecto interno de EternalGrowth.

## Características
- Pantalla de carga animada
- Diseño responsive (mobile-first)
- Componentes funcionales y hooks
- CSS puro y modular
- Paleta: Verde (#2F6B4F), Blanco (#FFFFFF), Rosado suave (#F4C2C2)
- Tipografías: Playfair Display (títulos), Poppins (texto)
- Sin dependencias externas salvo Google Fonts

## Estructura
- src/components: LoadingScreen, Navbar, Hero, About, Products, Testimonials, Contact, Footer
- src/styles/global.css
- src/components/components.css
- App.jsx, main.jsx

## Instalación y ejecución
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Ejecuta en local:
   ```bash
   npm run dev
   ```

## Nota
Todo el contenido y funcionalidades son demostrativos. Proyecto ficticio para práctica interna.
